# @artusjs/core

Core package of Artus

## Build

`@artusjs/core` write with TypeScript, so we need build source code to Node.js Module.

```bash
npm run build
# Or
yarn build
# Or
pnpm run build
```

## Run test case

`@artusjs/core` use `jest` for unit-test case.

```bash
npm run build
# Or
yarn build
# Or
pnpm run build
```
